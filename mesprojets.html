Projet GSB

La société Galaxy-Swiss-Bourdin (GSB) est un laboratoire pharmaceutique qui propose à ses visiteurs chargés de la communication auprès des praticiens un logiciel de gestion des visites afin de simplifier leur travail.

Le projet GSB consistait donc à développer en équipe, un site web de gestion des frais lors d’une visite. Cette application web a donc été réalisée à l'aide de Symfony sur VS Code et à une base de données à l'aide de Win'Design ainsi que de phpMyAdmin.

Projet FouDeSerie

FouDeSerie est une application web permettant de présenter différentes séries sur un site web développer à l'aide du Framework Symfony (sur Visual Studio Code) et d'une base de données MySql (sur phpMyAdmin).

Le projet étant de faire évoluer l'application avec différentes missions comme :

  • Ajout d'un formulaire permettant d'ajouter une série
  • Liker les séries
  • Modifier une série
  • Etc..
